An honest limitation, stated plainly. Photocatalytic H₂ rate can't be predicted to an exact number from literature data — the same material is reported with 100× different rates depending on nanostructure, morphology, and synthesis, none of which the source papers consistently record. So the photocatalysis tool does what the data can support honestly: it classifies performance tiers and screens "is this promising?" (ROC-AUC 0.72), and always shows the real published spread instead of a false-precision figure. For a chemist, an honest range beats a confident wrong number.
The credibility test that matters most. Several of Prof. Dutta's own published photocatalysts — g-C₃N₄, CdS, TiO₂, ZnO — are in the evidence base with their real measured H₂ rates. When you screen them here, the model places them where his lab actually found them. That agreement with hands-on experimental results is worth more than any single accuracy score.
